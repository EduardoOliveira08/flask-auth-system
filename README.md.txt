# Sistema de Login com Flask

Projeto de autenticação utilizando Flask e MySQL.

## Funcionalidades

- Cadastro de usuários
- Login
- Logout
- Sessão de usuário
- Dashboard protegido
- Senhas criptografadas

## Tecnologias

- Python
- Flask
- MySQL
- HTML/CSS

## Como executar

Instale as dependências:

```bash
pip install -r requirements.txt
```

Execute:

```bash
python app.py
```
